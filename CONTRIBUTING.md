# Contributing to InterQ

First off, thank you for considering contributing to InterQ! It's people like you who make this a great tool for job seekers worldwide.

### Reporting Bugs
- Check the issues tab to see if the bug has already been reported.
- If not, open a new issue with a clear title and detailed description of how to reproduce it.

### Suggesting Enhancements
- We love new ideas! Whether it's a new career path or a UI improvement, feel free to open an issue to discuss it.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the UI remains responsive and follows the existing design language (Tailwind + Framer Motion).
4. Maintain the "Privacy First" rule: Do not add external tracking or server-side storage for user recordings.
